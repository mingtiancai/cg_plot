# 1.计算几何基础数学
几何分为两种。一种是使用初等解析曲面，如平面/圆柱面等。另外一种使用自由曲面，如汽车车身/飞机机翼等，不能使用初等函数清楚表达。而是需要构建新的函数表示。

计算几何的核心在于应用，研究计算机环境中曲线曲面的表示与逼近。根据定义形状的几何信息建立相应的曲线曲面方程，即几何模型，并借助程序绘制这些曲面。

计算几何本身融合了微分几何，代数几何，数值计算，逼近论，拓扑学。

# 2.编译

```

```

```
xmake
```

运行指定程序
```
xmake run cg_plot
```
